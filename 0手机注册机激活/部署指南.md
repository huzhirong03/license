# 激活码生成器 - GitHub Pages 部署指南

## 📋 项目说明

这是一个Web版的激活码生成器，可以部署到GitHub Pages，实现手机/电脑随时访问使用。

**支持的程序：**
- 矢量转换工具 (R2V) - 使用 SHA256 + MD5 算法
- 宏嫖边工具 (VBA插件) - 使用自定义 SimpleHash 算法

**功能特点：**
- ✅ 密码保护，只有管理员可以访问
- ✅ 每次生成唯一激活码（使用随机盐值）
- ✅ 支持续费（再次生成新的激活码）
- ✅ 手机/电脑通用
- ✅ 一键复制

## 🚀 部署步骤

### 方法一：更新现有仓库（推荐）

如果你已经有 `https://github.com/huzhirong03/license` 仓库：

1. **打开仓库页面**
   - 访问：https://github.com/huzhirong03/license

2. **上传新文件**
   - 点击 "Add file" → "Upload files"
   - 将以下文件/文件夹拖拽上传：
     ```
     index.html
     css/style.css
     js/generator.js
     README.md
     部署指南.md
     ```

3. **或者使用 Git 命令行**
   ```bash
   # 克隆仓库
   git clone https://github.com/huzhirong03/license.git
   cd license
   
   # 复制新文件（替换旧文件）
   # 将 0手机注册机激活 文件夹内的所有文件复制到仓库根目录
   
   # 提交更改
   git add .
   git commit -m "更新激活码生成器 v2.0：支持VBA插件和R2V"
   git push origin main
   ```

4. **启用 GitHub Pages**
   - 进入仓库 Settings → Pages
   - Source 选择 "main" 分支
   - 点击 Save
   - 等待几分钟后访问：`https://huzhirong03.github.io/license/`

### 方法二：创建新仓库

1. **登录 GitHub**
   - 访问：https://github.com
   - 登录你的账号

2. **创建新仓库**
   - 点击右上角 "+" → "New repository"
   - Repository name: `license-generator`（或其他名称）
   - 设置为 **Private**（私有，只有你能看到）
   - 点击 "Create repository"

3. **上传文件**
   - 点击 "uploading an existing file"
   - 拖拽上传所有文件（保持目录结构）
   - 点击 "Commit changes"

4. **启用 GitHub Pages**
   - 进入 Settings → Pages
   - Source 选择 "main" 分支
   - 点击 Save

5. **访问页面**
   - 等待1-2分钟
   - 访问：`https://你的用户名.github.io/仓库名/`

## 📱 使用方法

1. 手机/电脑浏览器打开页面
2. 输入访问密码
3. 选择程序类型（R2V 或 VBA插件）
4. 输入用户提供的机器码
5. 选择激活时长
6. 点击生成激活码
7. 复制激活码发给用户

## ❓ 常见问题

### Q: 页面打不开？
A: 检查 GitHub Pages 是否已启用，等待1-2分钟再试

### Q: 激活码无效？
A: 检查密钥是否与程序中的密钥一致

### Q: 如何修改密码？
A: 编辑 `js/generator.js` 中的 `ACCESS_PASSWORD` 值

### Q: 可以离线使用吗？
A: 可以！将所有文件保存到本地，直接打开 index.html 即可
