# 激活码生成器 v2.0

私人激活码生成工具，支持多程序激活。

## 功能

- 🔐 密码保护
- 📱 手机/电脑通用
- 🔑 多程序支持
- 📋 一键复制
- 🔄 支持续费（每次生成唯一激活码）

## 支持的程序

| 程序名称 | 算法 | 状态 |
|---------|------|------|
| 矢量转换工具 (R2V) | SHA256 + MD5 | ✅ 已支持 |
| 宏嫖边工具 (VBA插件) | SimpleHash | ✅ 已支持 |

## 激活类型

### R2V 矢量转换工具
- 1分钟（测试用）
- 1个月
- 1年
- 永久

### VBA插件 宏嫖边工具
- 10秒（测试用）
- 1天
- 1个月（30天）
- 1年（365天）
- 永久

## 使用说明

1. 输入访问密码
2. 选择程序类型
3. 输入用户机器码（16位）
4. 选择激活时长
5. 点击"生成激活码"
6. 复制激活码发给用户
